# CL Crawler

> 这里主要是爬取小草上个人比较喜欢的项目,基本上爬取1024主站的其他映像都是同种结构,  
小草的网址是从百度搜的. 搜索免费可用的代理用了不少时间,许多ip质量不佳

* 墙内访问
> start_urls 爬取列表
num_page 列表每项搜索页面数量
* 最新代理ip池   
> num_IP  ip池数量

* user-agent池
* 防网址重复爬取
* 防内容重复
* 自定义关心内容
> rules 包含关键词且图片数量大于30
keep_if_exist_word 如果包含即爬取图片
save_dir 本地存储目录
